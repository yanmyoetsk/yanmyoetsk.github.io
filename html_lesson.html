HTML 初心者向けコース（レッスン1〜10）

レッスン1：HTMLとは何か

HTML は Webページを作るための言語です。
HTML は プログラミング言語ではありません。
Webページの「構造（こうぞう）」を作ります。 
<h1>Hello World</h1>
HTML：内容 / CSS：デザイン / JavaScript：動き

レッスン2：HTMLの基本構造

HTML には 必ず決まった基本の形があります。
<head> は 設定、<body> は 表示される内容です。 
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    Hello HTML
  </body>
</html>

レッスン3：文字と見出し

見出しは <h1>、文章は <p> を使います。 
<h1>Welcome</h1>
<p>This is my first web page.</p>

レッスン4：リンクと画像

リンクは <a>、画像は <img> を使います。 
<a href="https://example.com">Go to site</a>
<img src="image.jpg" alt="sample image">
alt は 画像が表示できない時の説明文です

レッスン5：リスト（一覧）

リストは 複数の項目を見やすく表示します。 
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

レッスン6：テーブル（表）

テーブルは 表（ひょう）を作る時に使います。 
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Taro</td>
    <td>25</td>
  </tr>
</table>

レッスン7：フォーム（基本）

フォームは ユーザーに入力してもらう画面です。 
<form>
  Name: <input type="text"><br>
  Email: <input type="email"><br>
  <button>Send</button>
</form>

レッスン8：フォーム要素（詳細）

入力内容によって type を使い分けます。 
<input type="password">

<input type="radio" name="g"> Male
<input type="radio" name="g"> Female

<input type="checkbox"> HTML

レッスン9：HTMLレイアウト

レイアウトは ページの配置（はいち）です。
HTML5 では 意味が分かるタグを使います。 
<header>
  <h1>My Website</h1>
</header>

<main>
  <p>Main content</p>
</main>

<footer>
  <p>© 2026</p>
</footer>

レッスン10：総合練習（自己紹介ページ）

今まで学んだタグを使って、
自己紹介ページを作りましょう。 
<h1>My Profile</h1>
<p>Name: Taro</p>

<h2>Hobby</h2>
<ul>
  <li>Music</li>
  <li>Travel</li>
</ul>

<form>
  <textarea></textarea><br>
  <button>Send</button>
</form>
