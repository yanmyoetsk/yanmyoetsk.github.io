HTML 初心者向け研修教材（タグ用途・属性 詳細解説付き）

レッスン1：HTMLとは何か

HTML は Webページの「構造」を定義する言語です。 文字・画像・リンクなどが「何であるか」をブラウザに伝えます。
主なタグ
<h1>：最も重要な見出し
<p>：文章（段落） 
<h1>Hello World</h1>
<p>This is my first HTML page.</p>
【演習】 「HTMLの勉強を始めました」という文章を p タグで表示してください。 
<p>HTMLの勉強を始めました</p>

レッスン2：HTMLの基本構造

HTML は必ず決まった構造を持ちます。 この構造は **全てのWebページの土台** です。
タグの役割
<html>：HTML文書全体
<head>：設定情報（画面には表示されない）
<body>：画面に表示される内容 
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>My Page</title>
</head>
<body>
Hello
</body>
</html>
【演習】 title を「HTML 研修ページ」に設定してください。 
<title>HTML 研修ページ</title>

レッスン3：見出しと文章

見出しタグは文書構造を表します。 見た目目的で使ってはいけません。
見出しタグ
<h1>：ページの主題（1つだけ）
<h2>：章タイトル
<p>：本文 
<h1>Company Profile</h1>
<h2>About Us</h2>
<p>We are IT company.</p>
【演習】 h2 と p を使って自己紹介を書いてください。 
<h2>About Me</h2>
<p>I am a beginner engineer.</p>

レッスン4：リンクと画像

リンクはページ移動、画像は視覚情報を提供します。
重要な属性
href：リンク先URL
src：画像ファイル
alt：画像説明（必須） 
<a href="https://google.com">Google</a>
<img src="photo.jpg" alt="社員写真">
【演習】 自分の会社サイトへのリンクを作ってください。 
<a href="https://example.com">Company Site</a>

レッスン5：リスト

複数の項目を整理して表示するために使います。
<ul>：順番なし
<ol>：順番あり
<li>：項目 
<ul>
<li>HTML</li>
<li>CSS</li>
</ul>
【演習】 好きな技術を3つリスト表示してください。 
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>

レッスン6：テーブル（表）

表は業務システム（勤怠・一覧画面）で多用されます。
<table>：表全体
<tr>：行
<th>：見出しセル
<td>：データセル 
<table border="1">
<tr><th>Name</th><th>Time</th></tr>
<tr><td>Taro</td><td>09:00</td></tr>
</table>
【演習】 名前と出勤時間の表を作ってください。

レッスン7：フォーム（基本）

フォームはユーザーからデータを受け取ります。
<form>：入力全体
<input>：入力欄
type="text"：文字入力 
<form>
<input type="text">
</form>
【演習】 名前入力フォームを作成してください。

レッスン8：フォーム要素（詳細）

入力内容に合った type を使うことが重要です。
email：メール形式チェック
password：非表示入力
radio：1つ選択
checkbox：複数選択
select：選択肢
textarea：長文 
<input type="email">
<input type="password">
【演習】 email と submit を持つフォームを作成してください。

レッスン9：HTMLレイアウト

意味を持つタグで構造を表現します。
<header>：ヘッダー
<main>：メイン内容
<footer>：フッター 
<header>Header</header>
<main>Content</main>
<footer>Footer</footer>

レッスン10：総合演習

これまで学んだタグをすべて使います。
【演習】 自己紹介ページを作成してください。 
<h1>My Profile</h1>
<p>I am learning HTML.</p>
<ul>
<li>HTML</li>
</ul>
<form>
<textarea></textarea>
</form>
