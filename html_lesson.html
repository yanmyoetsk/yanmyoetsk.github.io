HTML 初心者向け完全コース（レッスン1〜10）

レッスン1：HTMLとは何か

HTML は Webページを作るための言語です。
HTML は プログラミング言語ではありません。
Webページの 文字・画像・リンクなどの「構造（こうぞう）」を作ります。
ブラウザ（Chrome など）は HTML を読み取って画面に表示します。
HTML例
<h1>Hello World</h1>
・HTML：内容と構造
・CSS：デザイン（色・レイアウト）
・JavaScript：動き

レッスン2：HTMLの基本構造

HTML には 必ず決まった基本の形があります。
この形は すべての HTML ファイルで共通です。
<head> は 設定、<body> は 表示される内容です。 
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    Hello HTML
  </body>
</html>
・<title> は ブラウザのタブに表示される名前
・画面に見えるのは <body> の中だけ

レッスン3：文字と見出し

Webページに文字を書くには タグを使います。
見出しは <h1>、文章は <p> を使います。
見出しは 内容のタイトルとして使います。 
<h1>Welcome</h1>
<p>This is my first web page.</p>
・<h1>〜<h6> まである
・<h1> は ページで一番大事

レッスン4：リンクと画像

リンクは 他のページへ移動するためのものです。
画像は 写真やイラストを表示するために使います。 
<a href="https://example.com">Go to site</a>

<img src="image.jpg" alt="sample image">
・href：リンク先のURL
・src：画像ファイル名
・alt：画像の説明（とても重要）

レッスン5：リスト（一覧表示）

リストは 複数の項目を整理して表示できます。
メニューや機能一覧でよく使います。 
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
・<ul>：点（・）のリスト
・<li>：リストの1項目

レッスン6：テーブル（表）

テーブルは 表（ひょう）を作る時に使います。
勤怠管理や一覧画面でよく使われます。 
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Taro</td>
    <td>25</td>
  </tr>
</table>
・tr：行（ぎょう）
・th：見出し
・td：データ

レッスン7：フォーム（基本）

フォームは ユーザーに入力してもらう画面です。
問い合わせ・ログイン・業務システムで使います。 
<form>
  Name: <input type="text"><br>
  Email: <input type="email"><br>
  <button>Send</button>
</form>
入力したデータは サーバー（PHPなど）に送ります

レッスン8：フォーム要素（詳細）

入力内容によって type を使い分けます。
正しい type を使うと 入力ミスを防げます。 
<input type="password">

<input type="radio" name="g"> Male
<input type="radio" name="g"> Female

<input type="checkbox"> HTML
・radio：1つだけ選択
・checkbox：複数選択

レッスン9：HTMLレイアウト

レイアウトは ページの配置（はいち）です。
HTML5 では 意味が分かるタグを使います。 
<header>Header</header>
<main>Main Content</main>
<footer>Footer</footer>
見た目の調整は CSS で行います

レッスン10：総合練習（自己紹介ページ）

今まで学んだタグをすべて使って、
簡単な 自己紹介ページを作りましょう。 
<h1>My Profile</h1>
<p>Name: Taro</p>

<h2>Hobby</h2>
<ul>
  <li>Music</li>
  <li>Travel</li>
</ul>

<form>
  <textarea></textarea><br>
  <button>Send</button>
</form>
