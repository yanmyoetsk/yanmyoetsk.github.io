HTML 完全初心者向け講座（全10回）

Lesson 1：HTMLとは何か

HTML（HyperText Markup Language）は、 Webページの構造を作るための言語です。 プログラミング言語ではありません。

主な役割

  • 文章を表示する
  • 見出し・画像・リンクを配置する
  • ページの骨組みを作る

Examples（5例）

<h1>Hello HTML</h1>
<p>これは文章です</p>
<p>HTMLは構造を作ります</p>
<h2>見出し</h2>
<p>CSSはデザイン</p>
Exercise：
「Hello World」と表示するHTMLを書いてみましょう。 
<p>Hello World</p>
Summary：
HTMLはWebページの「内容」と「構造」を作る言語。

Lesson 2：HTMLの基本構造

HTMLファイルは決まった構造を持っています。

主要タグと役割

  • <html>：HTML全体
  • <head>：設定情報
  • <body>：画面表示部分

Examples（5例）

<!DOCTYPE html>
<html></html>

<head>
<title>My Page</title>
</head>

<body>Hello</body>

<meta charset="UTF-8">

<title>HTML Lesson</title>
HTMLの基本構造を書いてみましょう。 
<!DOCTYPE html>
<html>
<head></head>
<body></body>
</html>
HTMLは必ず head と body を持つ。

Lesson 3：見出しと文章

文章は <p>、見出しは <h1>〜<h6> を使います。

タグ説明

  • <h1>：一番大きな見出し
  • <p>：段落

Examples（5例）

<h1>Title</h1>
<h2>Sub Title</h2>
<p>説明文</p>
<h3>Section</h3>
<p>内容</p>
見出しと文章を1つずつ書いてみましょう。 
<h1>My Page</h1>
<p>Welcome</p>
見出しは構造、文章は内容。

Lesson 4：リンクと画像

リンクは <a>、画像は <img> を使います。

属性（properties）

  • href：リンク先
  • src：画像ファイル
  • alt：画像説明

Examples（5例）

<a href="https://google.com">Google</a>
<img src="a.jpg">
<img src="b.png" alt="sample">
<a href="page.html">Next</a>
<a href="#">Top</a>
画像とリンクを1つずつ作りましょう。 
<a href="https://example.com">Example</a>
<img src="sample.jpg" alt="sample">
href・src・alt は超重要。

Lesson 5：リスト

複数項目はリストで表示します。

Examples（5例）

<ul><li>HTML</li></ul>
<ol><li>One</li></ol>
<li>Item</li>
<ul><li>A</li><li>B</li></ul>
<ol><li>Step1</li></ol>
3つのリストを作りましょう。 
<ul>
<li>Apple</li>
<li>Orange</li>
<li>Banana</li>
</ul>
一覧表示はリスト。

Lesson 6：テーブル

表形式のデータに使います。

Examples（5例）

<table></table>
<tr></tr>
<td>Data</td>
<th>Header</th>
<table border="1"></table>
名前と年齢の表を作りましょう。 
<table border="1">
<tr><th>Name</th><th>Age</th></tr>
<tr><td>Taro</td><td>20</td></tr>
</table>
table / tr / td / th を覚える。

Lesson 7：フォーム基礎

フォームは入力画面です。

Examples（5例）

<form></form>
<input type="text">
<input type="email">
<button>Send</button>
<textarea></textarea>
名前入力フォームを作りましょう。 
<form>
<input type="text">
<button>Send</button>
</form>
フォーム＝ユーザー入力。

Lesson 8：フォーム要素（詳細）

用途に応じて type を使い分けます。

Examples（5例）

<input type="password">
<input type="checkbox">
<input type="radio">
<select><option>A</option></select>
<input type="date">
チェックボックスとラジオを作りましょう。 
<input type="checkbox">HTML
<input type="radio" name="g">Male
<input type="radio" name="g">Female
type を覚えるとフォームは自由自在。

Lesson 9：HTMLレイアウト

HTML5は意味のあるタグを使います。

Examples（5例）

<header></header>
<nav></nav>
<main></main>
<section></section>
<footer></footer>
基本レイアウトを書きましょう。 
<header>Header</header>
<main>Main</main>
<footer>Footer</footer>
意味を意識したタグを使う。

Lesson 10：総合練習

今までの内容を全部使います。

Examples（5例）

<h1>Profile</h1>
<ul><li>Hobby</li></ul>
<img src="me.jpg">
<form><input></form>
<footer>©2026</footer>
自己紹介ページを作りましょう。 
<h1>My Profile</h1>
<p>Name: Taro</p>
<ul><li>Music</li></ul>
HTMLの基本はこれで完成。
