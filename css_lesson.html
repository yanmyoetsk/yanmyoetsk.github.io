CSS 3時間ワークショップ

Lecture + CSS code + HTML code + Live example

① CSSとは？

説明: CSSはWebページの見た目を整える言語です。色、文字サイズ、レイアウトなどを指定します。

講義ポイント: HTMLは内容、CSSはデザインを担当します。

Live Example

この文字は赤です。

CSSコード

p {
  color: red;
}

HTMLコード

<p>この文字は赤です。</p>

② セレクタ（Selector）

セレクタは「どの要素にスタイルを当てるか」を指定します。

1. 要素セレクタ

このpタグは青色です。

p {
  color: blue;
}
<p>このpタグは青色です。</p>

2. クラスセレクタ

この文字は青色（class）です。

.blue-text {
  color: blue;
}
<p class="blue-text">この文字は青色（class）です。</p>

3. IDセレクタ

この文字は赤色(ID)で太字です。

#special {
  color: red;
  font-weight: bold;
}
<p id="special">この文字は赤色(ID)で太字です。</p>

4. 子孫セレクタ

div内のpは紫です。

.parent p {
  color: purple;
}
<div class="parent"><p>div内のpは紫です。</p></div>

5. 疑似クラス

マウスをのせるとオレンジになります。

.hover-demo:hover {
  color: orange;
}
<p class="hover-demo">マウスをのせるとオレンジになります。</p>

③ Box Model

HTML要素は「箱」です。Content・Padding・Border・Marginの4つで構成されています。

1. 基本例

margin + padding + border の例
.box-demo {
  margin: 20px;
  padding: 20px;
  border: 4px solid black;
  background: #ecf0f1;
}
<div class="box-demo">margin + padding + border の例</div>

2. content-box（デフォルト）

width + padding + border = 実際のサイズ
.content-box {
  width: 200px;
  padding: 20px;
  border: 5px solid blue;
}
<div class="content-box">width + padding + border = 実際のサイズ</div>

3. border-box

box-sizing:border-box を使うとサイズ計算が簡単です。
.border-box {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 5px solid red;
}
<div class="border-box">box-sizing:border-box</div>

⑦ レスポンシブデザイン

画面サイズに応じてレイアウトを変えます。

@media (max-width: 768px){
  .flex-container { flex-direction: column; }
  .grid-container { grid-template-columns: 1fr; }
}
