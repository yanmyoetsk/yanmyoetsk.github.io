CSS 3時間 勉強会（解説付き＋コード例）

Lecture + Example + Notes + CSS Code

① CSSとは？

説明: CSSはWebページのデザインを変える言語です。色、文字サイズ、レイアウトなどをコントロールします。

講義ポイント: HTMLは内容を作り、CSSは見た目を作る。

例:

この文字は赤です。

p {
  color: red;
}

② セレクタ（Selector）

説明: セレクタは「どこを変えるか」を指定します。

講義ポイント: HTML要素、class、id、子孫、擬似クラスなど。

1. 要素セレクタ

このpタグは青になります。

p {
  color: blue;
}

2. クラスセレクタ

この文字は青色（class）です。

.blue-text {
  color: blue;
}

3. IDセレクタ

この文字は赤色（ID）で太字です。

#special {
  color: red;
  font-weight: bold;
}

4. 子孫セレクタ

div内のpタグは紫になります。

.parent p {
  color: purple;
}

5. 疑似クラス

マウスをのせるとオレンジになります。

.hover-demo:hover {
  color: orange;
}

6. 詳細度（Specificity）

優先順位: inline style > ID > class > element

③ Box Model（はこの仕組み）

説明: HTML要素はすべて箱です。content、padding、border、marginがあります。

講義ポイント: レイアウトを作るときに必須です。

1. 基本例

margin + padding + border の例
.box-demo {
  margin: 20px;
  padding: 20px;
  border: 4px solid black;
  background: #ecf0f1;
}

2. content-box（デフォルト）

width + padding + border = 実際のサイズ
.content-box {
  width: 200px;
  padding: 20px;
  border: 5px solid blue;
}

3. border-box

box-sizing:border-box を使うとサイズ計算が簡単です。
.border-box {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 5px solid red;
}

⑤ Flexbox

Flexboxは要素を効率よく並べる方法です。

1
2
3
.flex-container {
  display: flex;
  gap: 20px;
}

.flex-item {
  flex: 1;
  padding: 20px;
  background: #3498db;
  color: white;
  text-align: center;
}

ブラウザを縮小すると縦並びになります（レスポンシブ対応）。

⑥ Grid

B
C
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

.grid-item {
  background: #9b59b6;
  color: white;
  padding: 20px;
  text-align: center;
}

2次元レイアウトに便利です。

⑦ レスポンシブデザイン

画面サイズに応じてレイアウトを変えます。

@media (max-width: 768px) {
  .flex-container {
    flex-direction: column;
  }
  .grid-container {
    grid-template-columns: 1fr;
  }
}

スマホでも見やすくなります。
