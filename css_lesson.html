CSS 3時間 勉強会（解説付き）

Lecture + Example + Notes

① CSSとは？

説明: CSSはWebページのデザインを変える言語です。色、文字サイズ、レイアウトなどをコントロールします。

講義ポイント: HTMLは内容を作り、CSSは見た目を作る。

例:

この文字は赤です。

講義で伝える: 「pタグの文字色を赤にしました。」

② セレクタ（Selector）

説明: セレクタは「どこを変えるか」を指定します。

講義ポイント: HTML要素、class、id、子孫、擬似クラスなど。

1. 要素セレクタ

このpタグは青になります。

2. クラスセレクタ

この文字は青色（class）です。

3. IDセレクタ

この文字は赤色（ID）で太字です。

4. 子孫セレクタ

div内のpタグは紫になります。

5. 疑似クラス

マウスをのせるとオレンジになります。

6. 詳細度（Specificity）

優先順位: inline style > ID > class > element

講義で伝える: 「CSSではどのルールが優先されるかを理解することが重要です。」

③ Box Model（はこの仕組み）

説明: HTML要素はすべて箱です。箱にはcontent、padding、border、marginがあります。

講義ポイント: レイアウトを作るときに必須です。

1. 基本例

margin + padding + border の例

2. content-box（デフォルト）

width + padding + border = 実際のサイズ

3. border-box

box-sizing:border-box を使うとサイズ計算が簡単です。
