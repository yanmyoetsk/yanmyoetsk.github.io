HTML 初心者向け研修教材（講師用・完成版）

レッスン1：HTMLとは何か

HTML は Webページを作るための言語です。
正確には「見た目」ではなく、ページの構造を定義します。
プログラミング言語ではなく、マークアップ言語です。

例①：見出しを表示する

<h1>Hello World</h1>

例②：文章を表示する

<p>This is HTML.</p>

例③：見出し＋文章

<h1>Title</h1>
<p>Description</p>
HTML：構造 / CSS：デザイン / JavaScript：動き （役割分担が非常に重要です）
【演習】 「Welcome to HTML」という文字を見出しとして表示してください。
【解答例】 
<h1>Welcome to HTML</h1>

レッスン2：HTMLの基本構造

HTML には必ず決まった基本構造があります。 この構造を守らないと、正しく表示されない原因になります。

例①：最小構造

<!DOCTYPE html>
<html>
<body>Hello</body>
</html>

例②：head と title

<head>
<title>My Page</title>
</head>

例③：文字コード指定

<meta charset="UTF-8">
日本語ページでは UTF-8 の指定は必須です。
【演習】 title を「HTML 基礎研修」に設定してください。 
<head>
<title>HTML 基礎研修</title>
</head>

レッスン3：見出しと文章

見出しタグは文書構造を表します。 サイズ目的で使ってはいけません。 
<h1>Main Title</h1>
<h2>Sub Title</h2>
<p>Explanation text</p>

<p>Paragraph 1</p>
<p>Paragraph 2</p>
h1 は原則 1ページに1つです。
【演習】 自己紹介として、h2 と p を使って文章を書いてください。 
<h2>About Me</h2>
<p>I am learning HTML.</p>

レッスン4：リンクと画像

Webの最大の特徴は「リンク」です。 画像には必ず alt 属性を付けます。 
<a href="https://example.com">Example</a>
<img src="photo.jpg" alt="sample photo">
<a href="https://google.com">
<img src="logo.png" alt="Google">
</a>
【演習】 Google へのリンクを作成してください。 
<a href="https://google.com">Google</a>

レッスン5：リスト

リストはメニューや項目表示で頻繁に使います。 
<ul>
<li>Apple</li>
</ul>

<ol>
<li>Step 1</li>
</ol>

<ul>
<li>Fruit
  <ul><li>Apple</li></ul>
</li>
</ul>
【演習】 好きな食べ物を3つ、ul で作成してください。 
<ul>
<li>Pizza</li>
<li>Sushi</li>
<li>Cake</li>
</ul>

レッスン6：テーブル（表）

業務システムでは最も重要なタグの一つです。 
<table border="1">
<tr><td>A</td></tr>
</table>

<tr><th>Name</th><th>Age</th></tr>

<tr><td>Taro</td><td>20</td></tr>
【演習】 名前と出勤時間の表を作成してください。 
<table border="1">
<tr><th>Name</th><th>Time</th></tr>
<tr><td>Taro</td><td>09:00</td></tr>
</table>

レッスン7：フォーム（基本）

フォームはユーザー入力を受け取るために使います。 
<input type="text">
<button>Send</button>
<form>
<input type="text">
</form>
【演習】 名前入力フォームを作成してください。 
<form>
<input type="text">
</form>

レッスン8：フォーム要素（詳細）

入力内容に応じて type を正しく使うことが重要です。 
<input type="email">
<input type="password">

<input type="radio">
<input type="checkbox">

<select><option>Nagoya</option></select>
<textarea></textarea>
【演習】 email と submit ボタンを持つフォームを作成してください。 
<form>
<input type="email">
<input type="submit">
</form>

レッスン9：HTMLレイアウト

HTML5 では意味を持つレイアウトタグを使います。 
<header>Header</header>
<main>Main</main>
<footer>Footer</footer>
【演習】 header・main・footer を使ってページ構造を作成してください。 
<header>Header</header>
<main>Content</main>
<footer>Footer</footer>

レッスン10：総合演習

これまで学んだ内容をすべて使います。
【演習】 自己紹介ページを作成してください。 
<h1>My Profile</h1>
<p>I am learning HTML.</p>
<ul>
<li>HTML</li>
<li>CSS</li>
</ul>
<form>
<textarea></textarea>
</form>
