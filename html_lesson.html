HTML 完全初心者向け講座（全10回）

Lesson 1：HTMLとは何か

HTML（HyperText Markup Language）は、 Webページの構造を作るための言語です。 プログラミング言語ではありません。

主な役割

  • 文章を表示する
  • 見出し・画像・リンクを配置する
  • ページの骨組みを作る

Examples（5例）

<h1>Hello HTML</h1>
<p>これは文章です</p>
<p>HTMLは構造を作ります</p>
<h2>見出し</h2>
<p>CSSはデザイン</p>
Exercise：
「Hello World」と表示するHTMLを書いてみましょう。 
<p>Hello World</p>
Summary：
HTMLはWebページの「内容」と「構造」を作る言語。

Lesson 2：HTMLの基本構造

HTMLコメント（）

HTMLコメントは、
ブラウザに表示されないメモです。
コードの説明や一時的な無効化に使います。

コメントの書き方

<!-- これはコメントです -->
✔ ブラウザには表示されない
✔ ソースコード上でのみ見える

Examples（実務でよく使う例）

<!-- ヘッダー開始 -->
<header>
  <h1>My Site</h1>
</header>
<!-- ヘッダー終了 -->

<!-- TODO: 後で修正する -->
<p>仮の文章</p>

<!--
<p>このコードは一時的に無効</p>
-->
💡 実務ポイント：
・チーム開発で非常に重要
・将来の自分や他人のために書く
Summary：
HTMLコメントは「表示されない説明文」。
保守性・可読性を上げるために必須。
HTMLファイルは決まった構造を持っています。

主要タグと役割

  • <html>：HTML全体
  • <head>：設定情報
  • <body>：画面表示部分

Examples（5例）

<!DOCTYPE html>
<html></html>

<head>
<title>My Page</title>
</head>

<body>Hello</body>

<meta charset="UTF-8">

<title>HTML Lesson</title>
HTMLの基本構造を書いてみましょう。 
<!DOCTYPE html>
<html>
<head></head>
<body></body>
</html>
HTMLは必ず head と body を持つ。

Lesson 3：見出しと文章

文章は <p>、見出しは <h1>〜<h6> を使います。

タグ説明

  • <h1>：一番大きな見出し
  • <p>：段落

Examples（5例）

<h1>Title</h1>
<h2>Sub Title</h2>
<p>説明文</p>
<h3>Section</h3>
<p>内容</p>
見出しと文章を1つずつ書いてみましょう。 
<h1>My Page</h1>
<p>Welcome</p>
見出しは構造、文章は内容。

Lesson 4：リンクと画像

リンクは <a>、画像は <img> を使います。

属性（properties）

  • href：リンク先
  • src：画像ファイル
  • alt：画像説明

Examples（5例）

<a href="https://google.com">Google</a>
<img src="a.jpg">
<img src="b.png" alt="sample">
<a href="page.html">Next</a>
<a href="#">Top</a>
画像とリンクを1つずつ作りましょう。 
<a href="https://example.com">Example</a>
<img src="sample.jpg" alt="sample">
href・src・alt は超重要。

Lesson 6：テーブル

テーブルで使うタグの説明

テーブルは複数のタグをセットで使います。
それぞれの役割を正しく理解しましょう。
タグ 役割（用途） 説明 注意点
<table> 表全体 テーブル全体を囲む レイアウト目的で使わない
<tr> 行（row） 1行分のデータ 必ず table 内に書く
<th> 見出しセル 項目名・タイトル 通常は最上段または左列
<td> データセル 実際の値を表示 文字・数字どちらもOK
✔ th は見出し、td はデータ
✔ 構造を意識して書くと読みやすい
Summary：
テーブルは「情報を整理して伝える」ためのタグ。
構造を意識して正しく使う。
表形式のデータに使います。

Examples（5例）

<table></table>
<tr></tr>
<td>Data</td>
<th>Header</th>
<table border="1"></table>
名前と年齢の表を作りましょう。 
<table border="1">
<tr><th>Name</th><th>Age</th></tr>
<tr><td>Taro</td><td>20</td></tr>
</table>
table / tr / td / th を覚える。

Lesson 8：フォーム要素（詳細・実務編）

フォームは業務システムの入口です。 ログイン、ユーザー登録、問い合わせ、申請画面など ほぼすべてのWebシステムで使われます。

form タグの重要属性

  • action：送信先URL（PHP / API）
  • method：GET または POST
  • name：サーバー側で受け取るキー名
✔ 実務では POST が基本
✔ パスワード・個人情報は GET を使わない

① ユーザー登録フォーム（Registration Form）

用途：
新規ユーザーをシステムに登録する画面です。
会員登録・社員登録・管理画面作成で必須です。

Example：登録フォーム

<form action="register.php" method="post">

  ユーザー名：<br>
  <input type="text" name="username" required><br><br>

  メールアドレス：<br>
  <input type="email" name="email" required><br><br>

  パスワード：<br>
  <input type="password" name="password" required><br><br>

  性別：<br>
  <input type="radio" name="gender" value="male"> 男性
  <input type="radio" name="gender" value="female"> 女性
  <br><br>

  <button type="submit">登録</button>

</form>

詳細解説（講師向け説明）

  • action="register.php"
    　→ 入力内容を PHP に送る
  • method="post"
    　→ URLに表示されない（安全）
  • name 属性
    　→ PHP側で $_POST['username'] として受信
  • required
    　→ 未入力を防ぐ（HTML側の簡易バリデーション）
【Exercise】
・年齢（number）
・誕生日（date）
を追加してみましょう。 
年齢：<br>
<input type="number" name="age"><br><br>

誕生日：<br>
<input type="date" name="birthday">
登録フォームは「action・method・name」が最重要。

② ログインフォーム（Login Form）

用途：
システムにログインするための画面です。 勤怠管理、管理画面、社内システムで必須です。

Example：ログインフォーム

<form action="login.php" method="post">

  メールアドレス：<br>
  <input type="email" name="email" required><br><br>

  パスワード：<br>
  <input type="password" name="password" required><br><br>

  <button type="submit">ログイン</button>

</form>

詳細解説（業務視点）

  • ログインは最低限の項目
    　→ ID（メール）＋ パスワード
  • POST必須
    　→ GETは絶対NG
  • 実務では
    　→ PHPでパスワードハッシュを比較
⚠ 実務注意点：
HTMLだけではセキュリティは不十分。
必ずサーバー側でチェックする。
【Exercise】
「ログイン状態を保存する」チェックボックスを追加しましょう。 
<input type="checkbox" name="remember">
ログイン状態を保存する
ログインフォームは「安全第一」。

Lesson 10：総合Webページ設計（まとめ）

このレッスンでは、
Lesson 1〜9 で学んだすべての要素を使って
「1つの完成したWebページ」を設計します。
📌 実務では、HTMLは
「タグを知っている」ではなく
「どう組み合わせてページを作るか」が重要です。

■ 今回作るページの構成（設計）

  • ヘッダー（サイト名）
  • ナビゲーション（リンク）
  • メインコンテンツ（説明・画像・リスト）
  • お問い合わせフォーム
  • フッター（コピーライト）

Example 1：ページ全体のレイアウト（HTML5）

<header>
  <h1>My Company</h1>
</header>

<nav>
  <a href="#">Home</a> |
  <a href="#">Service</a> |
  <a href="#">Contact</a>
</nav>

<main>
</main>

<footer>
  © 2026 My Company
</footer>
✔ header / nav / main / footer は
ページ構造を明確にするための意味のあるタグです。

Example 2：見出しと文章（Lesson 3）

<h2>会社概要</h2>
<p>
私たちはWebシステム開発を行っています。
</p>
✔ 見出しで情報の区切りを作る ✔ pタグは文章の基本

Example 3：画像とリスト（Lesson 4・5）

<img src="office.jpg" alt="会社の写真">

<h3>主なサービス</h3>
<ul>
  <li>Webシステム開発</li>
  <li>業務システム</li>
  <li>保守・運用</li>
</ul>
✔ alt は必須（アクセシビリティ・SEO） ✔ サービス一覧はリストが最適

Example 4：表（Lesson 6）

<table border="1">
  <tr>
    <th>項目</th>
    <th>内容</th>
  </tr>
  <tr>
    <td>設立</td>
    <td>2020年</td>
  </tr>
</table>
✔ 表は「比較・一覧情報」に使う ✔ レイアウト目的では使わない（重要）

Example 5：お問い合わせフォーム（Lesson 7・8）

<form action="contact.php" method="post">

  お名前：<br>
  <input type="text" name="name" required><br><br>

  メール：<br>
  <input type="email" name="email" required><br><br>

  お問い合わせ内容：<br>
  <textarea name="message"></textarea><br><br>

  <button type="submit">送信</button>

</form>
✔ form / input / textarea / button ✔ action・method・name を必ず意識
