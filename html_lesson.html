HTML 初心者向け完全教材（演習・解答付き）

レッスン1：HTMLとは

HTML は Webページの構造（骨組み）を作る言語です。 
<h1>Hello World</h1>
<p>This is HTML</p>
<h1>Title</h1>
<p>Description</p>
【練習】
「Welcome」という文字を見出しで表示してください。
【解答例】 
<h1>Welcome</h1>

レッスン2：HTML基本構造

HTML には必ず決まった基本構造があります。 
<!DOCTYPE html>
<html><body>Hi</body></html>

<head><title>My Page</title></head>

<meta charset="UTF-8">
【練習】
title に「HTML Practice」と設定してください。
【解答例】 
<head>
<title>HTML Practice</title>
</head>

レッスン3：見出しと文章

<h1>Main Title</h1>
<h2>Sub Title</h2>
<p>Text</p>

<p>Line 1</p>
<p>Line 2</p>
【練習】
h2 と p を使って自己紹介文を書いてください。
【解答例】 
<h2>About Me</h2>
<p>I am a beginner.</p>

レッスン4：リンクと画像

<a href="https://example.com">Example</a>
<img src="photo.jpg" alt="photo">
<a href="https://google.com">
<img src="logo.png">
</a>
【練習】
Google へのリンクを作成してください。
【解答例】 
<a href="https://google.com">Google</a>

レッスン5：リスト

<ul>
<li>Apple</li>
</ul>

<ol>
<li>Step 1</li>
</ol>

<ul>
<li>Fruit
  <ul><li>Apple</li></ul>
</li>
</ul>
【練習】
好きな食べ物を3つ ul で作ってください。
【解答例】 
<ul>
<li>Pizza</li>
<li>Sushi</li>
<li>Cake</li>
</ul>

レッスン6：テーブル

<table border="1">
<tr><td>A</td></tr>
</table>

<tr><th>Name</th></tr>

<tr><td>09:00</td><td>18:00</td></tr>
【練習】
名前と年齢の表を作ってください。
【解答例】 
<table border="1">
<tr><th>Name</th><th>Age</th></tr>
<tr><td>Taro</td><td>20</td></tr>
</table>

レッスン7：フォーム基本

<input type="text">
<button>Send</button>
<form>
<input>
</form>
【練習】
名前入力フォームを作ってください。 
<form>
<input type="text">
</form>

レッスン8：フォーム要素（詳細）

<input type="email">
<input type="password">

<input type="radio">
<input type="checkbox">

<select><option>Tokyo</option></select>
<textarea></textarea>
【練習】
email と submit を持つフォームを作ってください。 
<form>
<input type="email">
<input type="submit">
</form>

レッスン9：レイアウト

<header>Header</header>
<main>Main</main>
<footer>Footer</footer>
【練習】
header・main・footer を使って構造を作ってください。 
<header>Header</header>
<main>Content</main>
<footer>Footer</footer>

レッスン10：総合練習

<h1>My Profile</h1>
<ul><li>HTML</li></ul>
<form>
<textarea></textarea>
</form>
【練習】
自己紹介ページを作ってください。 
<h1>My Profile</h1>
<p>I am learning HTML.</p>
<form>
<textarea></textarea>
</form>
